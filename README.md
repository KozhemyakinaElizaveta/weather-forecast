# Weather Forecast

Приложение развернуто на [GitHub Pages](https://kozhemyakinaelizaveta.github.io/weather-forecast).

```
git clone https://github.com/KozhemyakinaElizaveta/weather-forecast.git
cd weather-forecast
npm ci
npm run dev
```
##### Стек технологий
- HTML
- Vue
- JavaScript
